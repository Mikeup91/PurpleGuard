## 🚀 Overview
PurpleGuard is an end-to-end security ecosystem engineered to run entirely on Android via Termux. It leverages Google Gemini 2.0 to bridge the gap between raw vulnerability scanning and intelligent, business-aware risk assessment. Unlike traditional tools, PurpleGuard features a **Neural Feedback Loop** that evolves its own "ammunition" based on the defenses it encounters.

## 🧠 The "Brain" Architecture
The platform is powered by four specialized core modules:

* **🔍 The Scout:** A recursive web spider that maps deep attack surfaces and extracts potential entry points.
* **💣 The Omnibus:** A high-velocity injection engine that delivers multi-vector payloads (SQLi, XSS, RCE).
* **⚖️ Hephaestus Judge:** An AI cognitive layer that performs real-time triage, assigning Risk Levels and Business Impact to every "kill."
* **🧬 RedBrain Mutator:** An adversarial AI agent that takes successful hits and evolves them into complex, obfuscated bypasses for WAF/IDS evasion.

## 📊 Dashboard & Visibility
PurpleGuard features a live **React-based Command Center** providing real-time telemetry:
- **Live Kill-Count:** Visual tracking of successful breaches.
- **AI Intelligence Feed:** Streamed executive summaries from the Hephaestus Judge.
- **Evolutionary Log:** View mutated payloads as they are generated by the RedBrain.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS (Mobile-Optimized)
- **Engine:** Python 3.11+, Requests, BeautifulSoup4
- **AI Integration:** Google Gemini 2.0 (via lightweight REST bridge)
- **Infrastructure:** Termux-Services (Background Daemons), Cronie, Tmux

## 📈 Scalability: Auto-Swarm Mode
The platform supports **Mass-Scale Reconnaissance**. By feeding a `targets.txt` into the Auto-Swarm orchestrator, the system autonomously cycles through discovery, attack, and AI-analysis phases across hundreds of domains while operating entirely in the background.
